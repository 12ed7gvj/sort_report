插入排序法 (Insertion Sort) 演示

插入排序是一種簡單的排序演算法。它將陣列分成已排序和未排序兩部分，每次從未排序部分取一個元素，插入到已排序部分的正確位置。

示例陣列: [5, 2, 4, 6, 1, 3]

回主頁
點擊"下一步"開始排序過程。

插入排序程式碼 (JavaScript)


function insertionSort(arr) {
    for (let i = 1; i < arr.length; i++) {
        let key = arr[i];
        let j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
    return arr;
}
