插入排序法 (Insertion Sort) 演示

插入排序是一種簡單的排序演算法。它將陣列分成已排序和未排序兩部分，每次從未排序部分取一個元素，插入到已排序部分的正確位置。

插入排序程式碼 (JavaScript)


function insertionSort(arr) {
    for (let i = 1; i < arr.length; i++) {
        let key = arr[i];
        let j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
    return arr;
}

演算法原理

  • 外層迴圈: 從第二個元素開始，逐一處理每個元素
  • 內層迴圈: 將當前元素與已排序部分的元素比較，向左移動大於當前元素的元素
  • 時間複雜度: O(n²)
  • 空間複雜度: O(1)
