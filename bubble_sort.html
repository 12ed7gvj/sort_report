氣泡排序法 (Bubble Sort) 演示

氣泡排序是一種簡單的排序演算法。它重複地走訪要排序的數列，一次比較兩個元素，如果它們的順序錯誤就把它們交換過來。

氣泡排序程式碼 (JavaScript)


function bubbleSort(arr) {
    const n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交換 arr[j] 和 arr[j+1]
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
    return arr;
}

演算法原理

  • 外層迴圈: 控制排序的輪數，每輪將最大的元素"冒泡"到末尾
  • 內層迴圈: 比較相鄰元素，如果順序錯誤則交換
  • 時間複雜度: O(n²)，當 n 增加時，執行時間約成長為 n²
  • 空間複雜度: O(1)
