選擇排序法 (Selection Sort) 演示

選擇排序是一種簡單的排序演算法。它將陣列分成已排序和未排序兩部分，每次從未排序部分選擇最小的元素，放到已排序部分的末尾。

選擇排序程式碼 (JavaScript)


function selectionSort(arr) {
    const n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        let minIndex = i;
        for (let j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        // 交換 arr[i] 和 arr[minIndex]
        [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
    }
    return arr;
}

演算法原理

  • 外層迴圈: 控制已排序部分的邊界，每次選擇一個位置
  • 內層迴圈: 在未排序部分中找到最小元素的索引
  • 交換: 將找到的最小元素與當前位置的元素交換
  • 時間複雜度: O(n²)
  • 空間複雜度: O(1)
